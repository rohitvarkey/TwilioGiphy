MMS Giphy translations!

This is a website you can use to send me (@rohitvarkey) GIFs on my phone! Just enter in any thing you want in the search box here. I use Giphy's translate API to enable this and Twilio for sending for sending out the MMS. The code can be found at [Github](https://github.com/rohitvarkey/TwilioGiphy).

This is a proof of concept app and will be taken down once my free trail of the Twilio API is done. Thanks Twilio! :)
